##### ERROR

Error: You cannot call "get" on a collection with no paths. Instead, check the "length" property first to verify at least 1 path exists.

<hr />

###### 발생

Vuetify 추가시 발생하는 오류

<br />

###### 원인

vue 버전이 안맞아서 발생

<br />

###### 해결

```javascript
yarn add vuetify
```

```js
// main.js
import Vuetify from 'vuetify'
import 'vuetify/dist/vuetify.min.css'

Vue.use(Vuetify)
```
