##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 정규표현식 공부 해야겠다..

```python
def solution(files):
    answer = []
    tmp = []
    for file in files:
        head, number = '', ''
        for f in file:
            if not f.isnumeric() and not number:
                head += f
            if f.isnumeric():
                number += f
            if not f.isnumeric() and number:
                break
        tmp.append([head, number, files.index(file)])
    tmp = sorted(tmp, key=lambda x : (x[0].lower(), int(x[1])))
    for i in tmp:
        answer.append(files[i[2]])
    return answer
```
