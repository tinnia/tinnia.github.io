###### 입출력

| s                | result  |
| ---------------- | ------- |
| `"110010101001"` | `[3,8]` |
| `"01110"`        | `[3,3]` |
| `"1111111"`      | `[4,1]` |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(s):
    cnt = i = 0
    while 1:
        cnt += s.count("0")
        s = format(len(s) - s.count("0"), 'b')
        i += 1
        if s == '1':
            break
    return [i, cnt]
```
