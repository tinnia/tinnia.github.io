###### 입출력

| n    | result |
| ---- | ------ |
| 78   | 83     |
| 15   | 23     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    ns = bin(n).count("1")
    while 1:
        n += 1
        if bin(n).count("1") == ns:
            break
    return n
```
