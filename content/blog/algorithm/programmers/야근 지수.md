###### 입출력

| works     | n    | result |
| --------- | ---- | ------ |
| [4, 3, 3] | 4    | 12     |
| [2, 1, 2] | 1    | 6      |
| [1,1]     | 3    | 0      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 단순 list를 heapify로 heap으로 바꾸기

```python
import heapq


def solution(n, works):
    if sum(works) <= n:
        return 0
    works = [-i for i in works]
    heapq.heapify(works)
    for i in range(n):
        tmp = heapq.heappop(works) + 1
        heapq.heappush(works, tmp)
    return sum([i ** 2 for i in works])
```
