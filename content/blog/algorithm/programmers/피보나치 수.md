###### 입출력

| n    | return |
| ---- | ------ |
| 3    | 2      |
| 5    | 5      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n] % 1234567
```
