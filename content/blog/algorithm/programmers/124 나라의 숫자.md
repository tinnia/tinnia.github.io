###### 입출력

| n    | result |
| ---- | ------ |
| 1    | 1      |
| 2    | 2      |
| 3    | 4      |
| 4    | 11     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    answer = ''
    rule = ['4', '1', '2']
    while n:
        n, m = divmod(n, 3)
        answer = rule[m] + answer
        if not m:
            n -= 1
    return answer
```
