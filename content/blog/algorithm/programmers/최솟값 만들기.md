###### 입출력

| A         | B         | answer |
| --------- | --------- | ------ |
| [1, 4, 2] | [5, 4, 4] | 29     |
| [1,2]     | [3,4]     | 10     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(A, B):
    answer = 0
    A.sort()
    B.sort(reverse=True)
    for i in range(len(A)):
        answer += A[i] * B[i]
    return answer
```
