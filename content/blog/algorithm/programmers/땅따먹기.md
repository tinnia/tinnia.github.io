###### 입출력

| land                            | answer |
| ------------------------------- | ------ |
| [[1,2,3,5],[5,6,7,8],[4,3,2,1]] | 16     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(land):
    for i in range(len(land)):
        if i == len(land) - 1:
            return max(land[i])
        for j in range(4):
            land[i+1][j] += max(land[i][:j] + land[i][j+1:])
```
