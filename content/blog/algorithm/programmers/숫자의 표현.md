###### 입출력

| n    | result |
| ---- | ------ |
| 15   | 4      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    answer = 0

    def func(idx, sum):
        nonlocal answer
        if sum >= n:
            if sum == n:
                answer += 1
            return
        func(idx + 1, sum + idx + 1)

    for i in range(1, n+1):
        func(i, i)
    return answer
```
