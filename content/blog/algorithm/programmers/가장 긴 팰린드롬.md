###### 입출력

| s       | answer |
| ------- | ------ |
| abcdcba | 7      |
| abacde  | 3      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def check(st):
    return st == st[::-1]


def solution(s):
    answer = 0
    for i in range(len(s)):
        for j in range(i, len(s) + 1):
            if check(s[i:j]):
                answer = max(answer, len(s[i:j]))
    return answer
```
