###### 입출력

| arr             | answer    |
| --------------- | --------- |
| [1,1,3,3,0,1,1] | [1,3,0,1] |
| [4,4,4,3,3]     | [4,3]     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(arr):
    answer = [arr[0]]
    for i in range(1, len(arr)):
        if arr[i] != answer[-1]:
            answer.append(arr[i])
    return answer
```
