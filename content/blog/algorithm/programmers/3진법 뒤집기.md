###### 입출력

| n    | result |
| ---- | ------ |
| 45   | 7      |
| 125  | 229    |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    tmp = ''
    while n:
        tmp += str(n % 3)
        n = n // 3

    answer = int(tmp, 3)
    return answer
```
