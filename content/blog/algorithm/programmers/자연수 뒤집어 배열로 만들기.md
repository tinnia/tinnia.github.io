###### 입출력

| n     | return      |
| ----- | ----------- |
| 12345 | [5,4,3,2,1] |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    answer = []
    n = str(n)
    for i in range(len(n) - 1, -1, -1):
        answer.append(int(n[i]))
    return answer
```
