| 16     | 4      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(num):
    answer = 0
    while True:
        if num == 1:
            return answer
        if num % 2 == 0:
            num //= 2
            answer += 1
        else:
            num = num * 3 + 1
            answer += 1

        if answer >= 500:
            if num != 1:
                answer = -1
                return answer
        if num == 1:
            return answer
```
