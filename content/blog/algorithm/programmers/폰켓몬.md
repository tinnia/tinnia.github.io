###### 입출력

| nums          | result |
| ------------- | ------ |
| [3,1,2,3]     | 2      |
| [3,3,3,2,2,4] | 3      |
| [3,3,3,2,2,2] | 2      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(nums):
    num = set(nums)
    return min(len(nums)//2, len(num))
```
