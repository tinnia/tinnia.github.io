###### 입출력

| n    | vertex                                                   | return |
| ---- | -------------------------------------------------------- | ------ |
| 6    | [[3, 6], [4, 3], [3, 2], [1, 3], [1, 2], [2, 4], [5, 2]] | 3      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n, edge):
    route = dict()
    for st, ed in edge:
        route.setdefault(st - 1, []).append(ed - 1)
        route.setdefault(ed - 1, []).append(st - 1)

    stack = [0]
    visit = [0] * n
    visit[0] = 1
    while stack:
        tmp = stack.pop(0)
        for i in route[tmp]:
            if visit[i] == 0:
                stack.append(i)
                visit[i] = visit[tmp] + 1
    return visit.count(max(visit))
```
