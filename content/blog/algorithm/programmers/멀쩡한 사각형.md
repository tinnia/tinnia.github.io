###### 입출력

| W    | H    | result |
| ---- | ---- | ------ |
| 8    | 12   | 80     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def gcd(n, m):
    if n % m == 0:
        return n
    return gcd(m, n % m)


def solution(w, h):
    return w * h - (w + h - gcd(w, h))
```
