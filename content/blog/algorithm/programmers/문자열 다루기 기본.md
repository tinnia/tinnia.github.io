###### 입출력

| s    | return |
| ---- | ------ |
| a234 | false  |
| 1234 | true   |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(s):
    if (len(s) == 4 or len(s) == 6) and s.isdigit():
        return True
    return False
```
