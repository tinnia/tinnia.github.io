###### 입출력

| new_id                          | result              |
| ------------------------------- | ------------------- |
| `"...!@BaT#*..y.abcdefghijklm"` | `"bat.y.abcdefghi"` |
| `"z-+.^."`                      | `"z--"`             |
| `"=.="`                         | `"aaa"`             |
| `"123_.def"`                    | `"123_.def"`        |
| `"abcdefghijklmn.p"`            | `"abcdefghijklmn"`  |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(new_id):
    answer = ''
    # 1
    new_id = new_id.lower()
    # 2
    st = "~!@#$%^&*()=+[{]}:?,<>/"
    for i in new_id:
        if i not in st:
            answer += i
    # 3
    while '..' in answer:
        answer = answer.replace('..', '.')
    # 4
    if answer[0] == '.':
        answer = answer[1:] if len(answer) > 1 else '.'
    if answer[-1] == '.':
        answer = answer[:-1]
    # 5
    if answer == '':
        answer += 'a'
    # 6
    if len(answer) >= 16:
        answer = answer[:15]
        if answer[-1] == '.':
            answer = answer[:-1]
    # 7
    while len(answer) < 3:
        answer += answer[-1]
    return answer
```
