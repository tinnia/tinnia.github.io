###### 입출력

| phone_number | return      |
| ------------ | ----------- |
| 01033334444  | *******4444 |
| 027778888    | *****8888   |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(phone_number):
    return "*"*(len(phone_number)-4) + phone_number[-4:]
```
