###### 입출력

| n    | t    | m    | p    | result           |
| ---- | ---- | ---- | ---- | ---------------- |
| 2    | 4    | 2    | 1    | 0111             |
| 16   | 16   | 2    | 1    | 02468ACE11111111 |
| 16   | 16   | 2    | 2    | 13579BDF01234567 |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n, t, m, p):
    answer = '0'
    nums = "0123456789ABCDEF"
    num = 1
    while len(answer) < t * m:
        tmp = num
        tmp_ans = ''
        while tmp:
            tmp_ans = nums[tmp % n] + tmp_ans
            tmp //= n
        answer += tmp_ans
        num += 1
    return answer[p - 1:t * m:m]
```
