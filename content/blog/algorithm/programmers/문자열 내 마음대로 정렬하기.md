###### 입출력

| strings           | n    | return            |
| ----------------- | ---- | ----------------- |
| [sun, bed, car]   | 1    | [car, bed, sun]   |
| [abce, abcd, cdx] | 2    | [abcd, abce, cdx] |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(strings, n):
    strings.sort()
    strings.sort(key=lambda x:x[n])
    return strings
```
