###### 입출력

| num  | return |
| ---- | :----: |
| 3    |  Odd   |
| 4    |  Even  |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(num):
    if num % 2:
        return "Odd"
    return "Even"
```
