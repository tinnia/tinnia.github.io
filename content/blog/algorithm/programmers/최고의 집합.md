###### 입출력

| n    | s    | result |
| ---- | ---- | ------ |
| 2    | 9    | [4, 5] |
| 2    | 1    | [-1]   |
| 2    | 8    | [4, 4] |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 단순 list를 heapify로 heap으로 바꾸기

```python
def solution(n, s):
    if n > s:
        return [-1]
    [x, y] = divmod(s, n)
    answer = [x] * n
    for i in range(y):
        answer[i] += 1
    return sorted(answer)
```
