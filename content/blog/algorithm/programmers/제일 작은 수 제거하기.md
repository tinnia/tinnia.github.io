###### 입출력

| arr       | return  |
| --------- | ------- |
| [4,3,2,1] | [4,3,2] |
| [10]      | [-1]    |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(arr):
    del arr[arr.index(min(arr))]
    if len(arr) == 0:
        return [-1]
    return arr
```
