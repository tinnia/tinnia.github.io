###### 입출력

| n    | m    | return  |
| ---- | ---- | ------- |
| 3    | 12   | [3, 12] |
| 2    | 5    | [1, 10] |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n, m):
    answer = []

    for i in range(n,0,-1):
        if n % i == 0 and m % i == 0:
            answer.append(i)
            break

    for i in range(m, m * n + 1):
        if i % m == 0 and i % n == 0:
            answer.append(i)
            break

    return answer
```
