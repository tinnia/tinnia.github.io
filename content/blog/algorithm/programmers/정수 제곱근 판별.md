###### 입출력

| n    | return |
| ---- | :----: |
| 121  |  144   |
| 3    |   -1   |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    if n ** 0.5 % 1:
        return -1
    else:
        return (int(n**0.5) + 1) ** 2
```
