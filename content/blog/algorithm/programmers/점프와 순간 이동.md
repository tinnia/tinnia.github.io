###### 입출력

| N    | result |
| ---- | ------ |
| 5    | 2      |
| 6    | 2      |
| 5000 | 5      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    ans = 0
    while n:
        if n % 2:
            n -= 1
            ans += 1
        else:
            n //= 2
    return ans
```
