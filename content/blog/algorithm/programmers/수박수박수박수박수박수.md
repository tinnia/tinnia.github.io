###### 입출력

| n    | return   |
| ---- | -------- |
| 3    | 수박수   |
| 4    | 수박수박 |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    answer = ''
    for i in range(n):
        if i % 2:
            answer += '박'
        else:
            answer += '수'
    return answer
```
