###### 입출력

| N    | answer |
| ---- | ------ |
| 123  | 6      |
| 987  | 24     |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(n):
    answer = 0
    n = str(n)
    for i in n:
        answer += int(i)

    return answer
```
