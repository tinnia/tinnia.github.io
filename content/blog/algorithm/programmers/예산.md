###### 입출력

| d           | budget | result |
| ----------- | ------ | ------ |
| [1,3,2,5,4] | 9      | 3      |
| [2,2,3,3]   | 10     | 4      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(d, budget):
    d.sort()
    cnt = 0
    for val in d:
        if val <= budget:
            budget -= val
            cnt += 1
    return cnt
```
