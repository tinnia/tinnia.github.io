| s      | result |

| baabaa | 1      |
| cdcd   | 0      |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 효율성이 0인 코드
* 아마 삭제하고 재정렬하는 시간 때문에 효율성에서 시간초과가 난 것 같다.

```python
def solution(s):
    i = 0
    while i < len(s) - 1:
        if s[i] == s[i+1]:
            s = s[:i] + s[i+2:]
            i = max(0, i - 1)
        else:
            i += 1
    if s:
        return 0
    else:
        return 1
```
