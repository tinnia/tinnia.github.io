###### 입출력

| s      | answer |
| ------ | ------ |
| ()()   | true   |
| (())() | true   |
| )()(   | false  |
| (()(   | false  |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(s):
    answer = True
    stack = []
    for i in range(len(s)):
        if s[i] == '(':
            stack.append(s[i])
        if stack:
            if s[i] == ')':
                stack.pop()
        else:
            answer = False
            break
    if stack:
        answer = False
    return answer
```
