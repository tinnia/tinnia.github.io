###### 입출력

| s           | return |
| ----------- | :----: |
| 1 2 3 4     |  1 4   |
| -1 -2 -3 -4 | -4 -1  |
| -1 -1       | -1 -1  |

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
def solution(s):
    lst = s.split()
    for i in range(len(lst)):
        lst[i] = int(lst[i])
    answer = str(min(lst)) + ' ' + str(max(lst))
    return answer
```

* 참고한 다른 코드

  def solution(s):
