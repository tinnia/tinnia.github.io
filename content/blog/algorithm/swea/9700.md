###### 출력

```sh
#1 YES
#2 NO
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for tc in range(1, int(input())+1):
    p,q = map(float,input().split())
    if (1-p)*q < p*(1-q)*q:
        print('#{} YES'.format(tc))
    else:
        print('#{} NO'.format(tc))
```
