###### 입력

```sh
2
3 5 6
6 8 20
```

###### 출력

```sh
#1 2
#2 3
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, int(input()) + 1):
    A, B, C = map(int, input().split())
    if A < B:
        print('#{} {}'.format(T, C // A))
    else:
        print('#{} {}'.format(T, C // B))
```
