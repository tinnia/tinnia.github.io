##### <h5 style="color:#C587AE;">💻 풀이</h5>

처음 풀이 - 시간초과

```python
for T in range(1, int(input()) + 1):
    N = int(input())
    lst = list(map(int, input().split()))
    Max = max(lst)
    for i in range(N-1):
        tmp = lst[i]
        for j in range(i+1,N):
            tmp += lst[j]
            if tmp > Max:
                Max = tmp
    print('#{} {}'.format(T, Max))
```

for T in range(1, int(input()) + 1):
    N = int(input())

    Max = -1111
    Sum = 0
    for i in range(N):
        Sum += lst[i]
        if Max < Sum:
            Max = Sum
        if Sum < 0:     # Sum이 0보다 작아지면 Reset 후 다음 index부터 다시 더하기
            Sum = 0
    print('#{} {}'.format(T, Max))
