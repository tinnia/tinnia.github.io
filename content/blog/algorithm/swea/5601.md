###### 입력

```sh

1
2
```

###### 출력

```sh
#1 1/1
#2 1/2 1/2
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, int(input())+1):
    N = int(input())
    print('#{}'.format(T), end=' ')
    for _ in range(N):
        print('1/{}'.format(N), end=' ')
    print()
```
