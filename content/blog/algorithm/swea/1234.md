##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, 11):
    N, nums = input().split()

    stack = []
    for num in nums:
        if not stack:
            stack.append(num)
        else:
            if num == stack[-1]:
                stack.pop()
            else:
                stack.append(num)
    print('#{}'.format(T), ''.join(stack))
```
