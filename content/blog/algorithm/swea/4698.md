###### 입력

```sh
2
3 10 30
7 1 1000000
```

###### 출력

```sh
#1 2
#2 43506
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
lst = [1] * 1000001
lst[0] = lst[1] = 0
tmp = []

for i in range(2, 1000001):
    if lst[i]:
        tmp.append(i)
        for j in range(2 * i, 1000001, i):
            lst[j] = 0

for T in range(1, int(input()) + 1):
    d, a, b = map(int, input().split())
    ans = 0
    for j in tmp:
        if a <= j <= b:
            if str(d) in str(j):
                ans += 1
        if j > b:
            break

    print('#{} {}'.format(T, ans))
```
