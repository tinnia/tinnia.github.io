###### 출력

```sh
#2 Exist
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, int(input()) + 1):
    st = list(input())
    flag = True
    for i in range(len(st) // 2):
        if st[i] != st[-1-i]:
            if st[i] != '*' and st[-1-i] != '*':
                flag = False
            break
    if flag:
        print('#{} Exist'.format(T))
    else:
        print('#{} Not exist'.format(T))
```
