###### 출력

```sh
#1 1
#2 2
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

단순 비교

```python
for T in range(1, int(input()) + 1):
    memory = input()
    cnt = 0
    tmp = "0"

    for i in range(len(memory)):
        if memory[i] != tmp:
            cnt += 1
            tmp = memory[i]
    print("#{} {}".format(T, cnt))
```
