###### 출력

```sh
#1 1 2 2 2 0
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, int(input()) + 1):
    N, Q = map(int, input().split())
    lst = [0] * N
    for i in range(1, Q + 1):
        L, R = map(int, input().split())
        for j in range(L-1, R):
            lst[j] = i
    print('#{}'.format(T), end=' ')
    print(*lst)
```
