###### 입력

```sh
5
4 0
4 30
4 47
5 31
5 62
```

###### 출력

```sh
#1 OFF
#2 OFF
#3 ON
#4 ON
#5 OFF
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, int(input()) + 1):
    N, M = map(int, input().split())
    tmp = format(M, 'b')
    if len(tmp) < N:
        print("#{} OFF".format(T))
    else:
        if "0" in tmp[-N:]:
            print('#{} OFF'.format(T))
        else:
            print('#{} ON'.format(T))
```
