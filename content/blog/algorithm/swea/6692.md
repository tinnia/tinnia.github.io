```sh
#1 30.700000
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for tc in range(1,int(input())+1):
    n=int(input())
    ans = 0
    for _ in range(n):
        a,b = map(float,input().split())
        ans+=a*b
    print('#{} {:.6f}'.format(tc, ans))
```
