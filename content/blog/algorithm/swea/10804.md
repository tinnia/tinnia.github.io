###### 입력

```sh
2
bdppq
qqqqpppbbd
```

###### 출력

```sh
#1 pqqbd
#2 bddqqqpppp
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
for T in range(1, int(input()) + 1):
    rule = {'b': 'd', 'd': 'b', 'p': 'q', 'q': 'p'}
    st = input()
    answer = ''
    for i in range(len(st)-1, -1, -1):
        answer += rule[st[i]]

    print('#{} {}'.format(T, answer))
```
