##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
K = int(input())
stack = []
for _ in range(K):
    tmp = int(input())
    if tmp == 0:
        stack.pop()
    else:
        stack.append(tmp)
print(sum(stack))
```
