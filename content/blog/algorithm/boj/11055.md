###### 입력

```sh
10
1 100 2 50 60 3 5 6 7 8
```

###### 출력

```sh
113
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = list(map(int, input().split()))
dp = [x for x in lst]
for i in range(N):
    for j in range(i):
        if lst[j] < lst[i]:
            dp[i] = max(dp[i], dp[j] + lst[i])

print(max(dp))
```
