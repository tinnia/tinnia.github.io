##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
n, m = map(int, input().split())
board = [input() for _ in range(n)]
a = b = 0

for i in range(n):
    if 'X' not in board[i]:
        a += 1

for j in range(m):
    if 'X' not in [board[i][j] for i in range(n)]:
        b += 1

print(max(a,b))
```
