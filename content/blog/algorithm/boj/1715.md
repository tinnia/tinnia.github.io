###### 입력

```sh
3
10
20
40
```

###### 출력

```sh
100
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
import heapq

N = int(input())
heap = []
for _ in range(N):
    heapq.heappush(heap, int(input()))

answer = 0
while len(heap) != 1:
    one = heapq.heappop(heap)
    two = heapq.heappop(heap)
    answer += one + two
    heapq.heappush(heap, one + two)

print(answer)
```
