###### 입력

```sh
5
1
5
3
1
2
```

###### 출력

```sh
3
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = [int(input()) for _ in range(N)]
lst.sort()
ans = 0
for i in range(N):
    ans += abs(i + 1 - lst[i])
print(ans)
```
