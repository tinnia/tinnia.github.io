###### 입력

```sh
5
6 3 2 10 -10
8
10 9 -5 2 3 4 5 -10
```

###### 출력

```sh
1 0 0 1 1 0 0 1
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = list(map(int, input().split()))
M = int(input())
card = list(map(int, input().split()))
lst.sort()
for i in card:
    st, ed = 0, N - 1
    while st <= ed:
        mid = (st + ed) // 2
        if i == lst[mid]:
            print(1, end=' ')
            break
        elif i > lst[mid]:
            st = mid + 1
        else:
            ed = mid - 1
    else:
        print(0, end=' ')
```
