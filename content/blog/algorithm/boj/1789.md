19

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 1~ n까지의 합이 n * (n + 1) // 2 인 것을 활용해서 풀었는데 사실 답이 맞을 줄 몰랐다.

```python
S = int(input())
n = 1
while n * (n + 1) / 2 <= S:
    n += 1
print(n - 1)
```
