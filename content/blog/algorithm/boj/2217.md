###### 출력


<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = sorted([int(input()) for _ in range(N)], reverse=True)
ans = [lst[i] * (i + 1) for i in range(N)]
print(max(ans))
```
