<a href="https://www.acmicpc.net/problem/2230"><small>문제 바로가기</small></a>

###### 입력

```sh
3 3
1
5
3
```

###### 출력

```sh
4
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N, M = map(int, input().split())
arr = sorted([int(input()) for _ in range(N)])
st, ed, ans = 0, 0, 2e9 + 1

while st < N and ed < N:
    tmp = arr[ed] - arr[st]
    if tmp >= M:
        ans = min(ans, tmp)
        st += 1
    else:
        ed += 1
print(ans)
```
