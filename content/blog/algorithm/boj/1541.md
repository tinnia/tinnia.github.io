###### 입력

```sh
55-50+40
```

###### 출력

```sh
-35
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 처음에 문제를 어떻게 풀어야 할지 고민했는데, 더할 수 있는 애들을 다 더해서 큰 수를 만들고 마이너스를 만날 때 마다 마이너스를 해주면 된당.

```python
lst = input().split('-')
ans = 0
for i in lst[0].split('+'):
    ans += int(i)
for i in lst[1:]:
    for j in i.split('+'):
        ans -= int(j)
print(ans)
```
