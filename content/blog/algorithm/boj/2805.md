###### 입력

```sh
4 7
20 15 10 17
```

###### 출력

```sh
15
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N, M = map(int, input().split())
tree = list(map(int, input().split()))
st, ed = 1, max(tree)

while st <= ed:
    mid = (st + ed) // 2
    ans = 0
    for i in tree:
        if i >= mid:
            ans += i - mid
    if ans >= M:
        st = mid + 1
    else:
        ed = mid - 1
print(ed)
```
