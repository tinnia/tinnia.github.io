<a href="https://www.acmicpc.net/problem/17298"><small>문제 바로가기</small></a>

###### 입력

```sh
4
3 5 2 7
```

###### 출력

```sh
5 7 7 -1
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = list(map(int, input().split()))
ans = [-1] * N
stack = [0]
i = 1
while stack and i < N:
    while stack and lst[stack[-1]] < lst[i]:
        ans[stack[-1]] = lst[i]
        stack.pop()
    stack.append(i)
    i += 1
print(*ans)
```
