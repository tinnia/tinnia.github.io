###### 입력

```sh
88
```

###### 출력

```sh
2
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = input()
tmp = '1' * len(N)

if len(N) == 1:
    print(1)
elif int(tmp) <= int(N):
    print(len(N))
else:
    print(len(N) - 1)
```
