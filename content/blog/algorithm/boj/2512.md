###### 입력

```sh

120 110 140 150
485
```

###### 출력

```sh
127
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = list(map(int, input().split()))
M = int(input())
st, ed = 0, max(lst)
while st <= ed:
    mid = (st + ed) // 2
    ans = 0
    for i in lst:
        ans += min(mid, i)
    if ans > M:
        ed = mid - 1
    else:
        st = mid + 1
print(ed)
```
