<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* [11053](https://www.acmicpc.net/problem/11053)번 문제랑 거의 비슷하다. 

```python
N = int(input())
lst = list(map(int, input().split()))
dp = [1] * N
for i in range(1, N):
    for j in range(i):
        if lst[i] < lst[j]:
            dp[i] = max(dp[i], dp[j] + 1)
print(N - max(dp))
```
