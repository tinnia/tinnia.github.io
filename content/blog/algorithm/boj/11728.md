###### 입력

```sh
2 2
3 5
2 9
```

###### 출력

```sh
2 3 5 9
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 그냥 해도 되지만, 투포인터를 연습하기 위해 투포인터로 풀어보았다.

```python
N, M = map(int, input().split())
A = list(map(int, input().split()))
B = list(map(int, input().split()))
ans = []
a = b = 0
while a <= N and b <= M:
    if a == N:
        ans += B[b:]
        break
    if b == M:
        ans += A[a:]
        break
    if A[a] >= B[b]:
        ans.append(B[b])
        b += 1
    elif B[b] > A[a]:
        ans.append(A[a])
        a += 1
print(*ans)
```
