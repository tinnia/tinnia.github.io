###### 입력

```sh
5
3 4
1 1
1 -1
2 2
3 3
```

###### 출력

```sh
1 -1
1 1
2 2
3 3
3 4
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N = int(input())
lst = []
for _ in range(N):
    x, y = map(int, input().split())
    lst.append((x, y))
lst.sort()
for i in lst:
    print(*i)
```
