###### 입력

```sh
4 2
1 1 1 1
```

###### 출력

```sh
3
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

* 투포인터의 대표적인 문제

```python
N, M = map(int, input().split())
lst = list(map(int, input().split()))

st = ed = ans = 0
while st <= ed <= len(lst):
    S = sum(lst[st:ed])
    if S == M:
        ans += 1
    if S <= M:
        ed += 1
    elif M < S and st < ed:
        st += 1
    else:
        st += 1
        ed += 1
print(ans)
```
