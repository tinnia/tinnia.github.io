<a href="https://www.acmicpc.net/problem/2559"><small>문제 바로가기</small></a>

###### 입력

```sh
10 2
3 -2 -4 -9 0 3 7 13 8 -3
```

###### 출력

```sh
21
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N, K = map(int, input().split())
lst = list(map(int, input().split()))
ans = tmp = sum(lst[:K])
for i in range(K, N):
    tmp += lst[i] - lst[i-K]
    ans = max(ans, tmp)
print(ans)
```
