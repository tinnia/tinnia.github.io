##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
from collections import deque

N = int(input())
Q = deque(list(range(1, N + 1)))
while len(Q) > 1:
   Q.popleft()
   Q.append(Q.popleft())
print(Q[0])
```
