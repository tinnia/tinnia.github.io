###### 입력

```sh
1 5 7
```

###### 출력

```sh
577
```

<br />

##### <h5 style="color:#C587AE;">💻 풀이</h5>

```python
N, K = map(int, input().split())
lst = list(map(int, input().split()))
ans = []


def func(x):
    if x > N:
        return
    ans.append(x)
    for i in lst:
        func(10 * x + i)


func(0)
print(max(ans))
```
