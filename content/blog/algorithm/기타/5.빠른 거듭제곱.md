### 빠른 거듭제곱

<br />

##### 기본

```python
# a ^ 11 = a ^ 1 + a ^ 2 + a ^ 8
def pow_advanced(a, b, mod):
    ans = 1
    while b > 0:
        if b % 2:
        	ans = ans * a % mod
        	a, b = a * a % mod, b // 2
    return ans

print(pow_advanced(3, 10000000, 1000000007))
print(pow(3, 10000000, 1000000007))	# 제일 빠름	
print(3 ** 10000000 % 1000000007)	# 가장 느림
```
