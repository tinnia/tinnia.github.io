### Factorial

<br />

##### 반복문

```python
def factorial(num):
    result = 1
    for i in range(1, num + 1):
        result *= i
    return result
```

<br />

###### 재귀

```python
def factorial(num):
	if num == 1:
        return 1
    return num * factorial(num - 1)
```

<br />

###### 라이브러리

```python
from math import factorial
print(factorial(5))
```
